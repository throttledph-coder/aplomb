FEATURE BREAKDOWN
Core MVP Features
text

1. Resume upload (PDF or plain text paste)
2. Job description entry (role, company, full description)
3. Resume parsing to extract: skills, experience, education, projects
4. Manual question input (free tier core feature)
5. AI answer generation personalized to resume + job description
6. Live session screen with question/answer display
7. Copy answer to clipboard with one click
8. Session save with all Q&A pairs
9. Post-session coaching report
10. Basic settings screen
Premium Features
text

1. Auto-listen mode (microphone captures interviewer)
2. Smart question detection (filter filler words, silence, non-questions)
3. Screen share invisibility toggle
4. System tray mode (app disappears from taskbar + screen)
5. Answer streaming (word by word, feels more natural to read)
6. (Pro = the live extras above; all preparation — sessions, resumes, export — is free + unlimited)
Nice-to-Have (Phase 3)
text

1. Multiple resume profiles (for different job types)
2. Company research auto-pull (scrape public info about the company)
3. Practice mode (mock interview with no real session)
4. Answer history within a session (navigate back through Q&A)
5. Favorite/bookmark specific answers
6. Export session as PDF
7. Confidence tips alongside answers
8. Follow-up question predictor
